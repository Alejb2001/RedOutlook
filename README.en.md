## Description

**RedOutlook** is a Reddit web client with a Microsoft Outlook-inspired interface. It allows you to browse Reddit posts and comments in a familiar and productive way, ideal for discreet use in work environments.

**Live Demo:** [redoutlook-production.up.railway.app](https://redoutlook-production.up.railway.app)

## Features

- **Outlook-style interface** - Familiar navigation with sidebar, post list, and reading pane
- **Subreddit browsing** - Explore different Reddit communities
- **Threaded comments** - View nested comments like on Reddit
- **Collapse threads** - Click the vertical bar to collapse/expand comment threads
- **Post sorting** - Hot, New, Top, Rising
- **Comment sorting** - Best, Top, New, Controversial, Old, Q&A
- **Customizable settings:**
  - Hide/show sorting tabs
  - Hide images and GIFs (replaces with text)
  - Show/hide NSFW content
- **Collapsible sidebar** - Hide the side panel for more space
- **Responsive design** - Works on different screen sizes
